This directory contains generated experiment outputs.

Baseline outputs are regenerated by:

```bash
python run_all.py --batch-size 8
```

The novelty modules can be rerun individually from `evaluation/`:

```bash
python -m evaluation.causal_attribution
python -m evaluation.set_size_fairness
python -m evaluation.counterfactual
```

Poster-ready figures (300 DPI, PDF + PNG) are produced by the helpers in
`scripts/`:

```bash
python scripts/make_poster_figures.py     # 4 novelty figures from CSVs
python scripts/regen_baseline_plots.py    # 3 baseline figures from cache
```

Headline writeups: [`BASELINE_README.md`](BASELINE_README.md) for baseline
numbers, [`NOVELTY_SUMMARY.md`](NOVELTY_SUMMARY.md) for the full 5-move
discussion of all three novelty modules.

Generated caches, logs, large raw counterfactual rows, and raw bootstrap
samples are ignored by git; the tracked CSV/Markdown files are the compact
report artifacts.
